为控件添加点击波纹效果

该方式使用的是系统提供的方式，需要在Android 5.0以上的版本才会生效。随着手机的更新换代，现在5.0的覆盖率已经挺多了，所以可以考虑使用这些效果。

方式一：使用系统申明的属性

先看效果图：

只需要在布局文件中设置控件的background属性值即可：

其中android:background="?attr/selectableItemBackgroundBorderless"设置后的效果是无边界的圆形扩展效果，android:background="?attr/selectableItemBackground"设置的是有边界的矩形扩展效果。

如果想更改默认波纹的颜色，可以通过修改主题样式：

上面设置的是系统定义的正常、激活、高亮3种状态下的效果颜色。

如果你好奇，仅仅添加了个background怎么就有了波纹效果呢？那我们就进一步看看这个?attr/selectableItemBackgroundBorderless到底是何方神圣？点击进去。

发现它定义在v7包values.xml的AppCompatTheme主题中，我们继续搜索一下这个selectableItemBackground可以看到为他赋值的地方：

可以看到，这是一个drawable资源，继续跟进，abc_item_background_holo_dark.xml的声明如下：

它是一个selector，所以会根据不同的状态，展现不同的资源，那么我们看看android:state_pressed="true"时的资源，点击abc_list_selector_background_transition_holo_dark

它是一个transition类型的drawable，这种类型的drawable对应的类是：TransitionDrawable

An extension of LayerDrawables that is intended to cross-fade between the first and second layer. To start the transition, call startTransition(int). To display just the first layer, call resetTransition().

It can be defined in an XML file with the element. Each Drawable in the transition is defined in a nested . For more information, see the guide to Drawable Resources.

TransitionDrawable可以在两个layer见进行渐变过渡。这就是使用一个background就实现了水波纹效果的原理。

方式二：使用RippleDrawable

还有一种方式，是5.0后提供的一种新的波纹效果的实现，直接定义ripple类型的drawable，它对应:RippleDrawable类，该类是API 21提供的，所以也只支持5.0以上的系统。

使用<ripple>标签，在drawable目录下定义相应的背景形状，然后也同样将其应用到控件的background上即可。这种波纹效果的具体的实现逻辑都在RippleDrawable中。
